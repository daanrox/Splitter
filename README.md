# Splitter API

## Descrição
Uma simples rota para fornecer informações dinâmicas para Split em uma gateway de pagamentos, o cliente terá controle total do usuário a ser splitado!

## Como Usar:

- Envie uma requisição GET para `/` e receberá os dados necessários para Split.

## Executando o Projeto

### Pré-requisitos
Certifique-se de ter o Node.js instalado.

### Instalação das Dependências
No terminal, execute o seguinte comando para instalar as dependências:

```bash
npm install
```

### Rodando o Projeto
Para iniciar o backend, utilize o comando: 

```bash
npm run start
```
Isso iniciará o servidor Node.js
